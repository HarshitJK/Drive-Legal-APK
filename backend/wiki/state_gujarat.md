# Gujarat — State Traffic Laws & Fine Schedule

## Overview
Gujarat follows the Motor Vehicles (Amendment) Act 2019. The state has a well-developed road network with active enforcement on national and state highways.

## Gujarat E-Challan System
- National portal: **echallan.parivahan.gov.in**
- Gujarat traffic portal: **gujarattraffic.gov.in**

## Key Fine Schedule (Gujarat)

| Violation | Fine |
|-----------|------|
| Speeding | ₹1,000 – ₹2,000 (first); ₹2,000 – ₹4,000 (repeat) |
| Helmet | ₹1,000 |
| Drunk Driving | ₹10,000 (first); ₹15,000 (repeat) — Note: Gujarat is a dry state |
| No Insurance | ₹2,000 (first); ₹4,000 (repeat) |
| No Licence | ₹5,000 |
| No PUC | ₹10,000 |
| Seatbelt | ₹1,000 |
| Mobile Phone | ₹5,000 |
| Red Light | ₹1,000 – ₹5,000 |

## Special Rules in Gujarat
- **Dry State**: Gujarat has prohibition on alcohol. Any alcohol consumption (even small) is illegal; drunk driving carries extra legal penalties beyond the MV Act.
- **Expressways**: Ahmedabad–Vadodara Expressway has 120 km/h limit for cars.
- **Surat Smart City**: ATCC cameras installed at major intersections.
