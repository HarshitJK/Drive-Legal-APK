# Karnataka — State Traffic Laws & Fine Schedule

## Overview
Karnataka follows the Motor Vehicles (Amendment) Act 2019. Bengaluru Traffic Police uses an extensive ATCC camera network across the city.

## Karnataka E-Challan System
- Bengaluru Traffic Police portal: **bengalurutrafficpolice.gov.in**
- National portal: **echallan.parivahan.gov.in**

## Key Fine Schedule (Karnataka Specific)

| Violation | Fine |
|-----------|------|
| Speeding | ₹1,000 – ₹2,000 (first); ₹2,000 – ₹4,000 (repeat) |
| Helmet | ₹1,000 |
| Drunk Driving | ₹10,000 (first); ₹15,000 (repeat) |
| No Insurance | ₹2,000 (first); ₹4,000 (repeat) |
| No Licence | ₹5,000 |
| No PUC | ₹10,000 |
| Seatbelt | ₹1,000 |
| Mobile Phone | ₹5,000; licence suspension possible |
| Red Light | ₹1,000 – ₹5,000 |
| Wrong Side | ₹1,000 – ₹5,000 |

## Special Rules in Karnataka
- **NICE Road / ORR Expressway**: 80 km/h speed limit; speed cameras throughout.
- **School zone**: 25 km/h mandatory. Strict enforcement in school hours.
- **No-Parking zones**: Towing in Bengaluru CBD. Charges: ₹700 (two-wheeler), ₹1,500 (car).
- **Signal-free corridors**: Special rules on elevated expressways.
