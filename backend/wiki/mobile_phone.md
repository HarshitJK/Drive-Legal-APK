# Mobile Phone While Driving — Section 184, Motor Vehicles Act 1988

## Overview
Using a mobile phone while driving (including texting, calling, or browsing) is a **dangerous driving** offence under **Section 184** of the Motor Vehicles Act, 1988.

## What is Prohibited
- Holding a mobile phone in hand while driving.
- Texting or browsing on a mobile device while driving.
- Using a phone at a red light (engine running = driving).
- Using earphones/headsets while driving (distraction offence in many states).

## What is Allowed
- Hands-free calling via Bluetooth headset (car hands-free kit).
- Using GPS/Maps when the phone is mounted on the dashboard (not held).

## Fines

| Offense | Fine |
|---------|------|
| First offense | ₹5,000 |
| Repeat offense | ₹10,000 |

## Additional Consequences
- Driving licence can be suspended.
- Court proceedings for causing accidents while using phone.
- Vehicle can be detained temporarily.

## State Variations
- **Karnataka**: ₹5,000 fine; licence suspension on repeat offences.
- **Delhi**: Active enforcement with cameras at intersections.
- **Tamil Nadu**: Heavy enforcement; challans issued via e-surveillance.
- **Maharashtra**: Dedicated phone-checking drives at toll booths.

## Key Points
- Even briefly glancing at your phone counts as using it while driving.
- Traffic cameras now detect mobile phone usage automatically.
- "I was using navigation" is not a valid defence if the phone was held.
